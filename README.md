# Cars Dealership - Full-Stack Web Application

## Repository Name
`agfzb-CloudAppDevelopment_Capstone`

## Project Name
**Cars Dealership - National Car Retailer Web Application**

## Overview
A full-stack responsive web application built for Cars Dealership, a national car retailer in the U.S.
Users can browse dealership branches, view dealer details, read and submit reviews.

## Technologies Used
- **Frontend:** React.js, HTML5, CSS3, Bootstrap
- **Backend:** Django (Python), Flask, Node.js
- **Database:** SQLite (Django models), MongoDB (reviews & dealers)
- **DevOps:** Docker, Kubernetes, IBM Cloud Code Engine, GitHub Actions (CI/CD)
